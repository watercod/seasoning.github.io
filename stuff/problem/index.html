Report a Problem


Use this to tell us about an issue that should be fixed. If you have a suggestion for an edit or a game request, please use the Submit Something form. Thanks.

A few notes:


Please don't just say "the games don't work". This doesn't really give me any info about the problem, so I won't be able to fix it.


If there is a graphical/audio glitch in one of the games, I am unable to fix this since I don't own the games on this site. If you do find an issue however, please do submit it, and I'll just add a note so that other people know about it.


If you are having an issue with a game loading, there are two things you should try first:

1: close all your open tabs/apps on your computer, and then go to the page again

2: if that still does not work, give it about 60 seconds to load. If the game is not flash, sometimes the server for the game will take a while to respond if it hasn't had any requests in a while.

If neither of those work, then feel free to fill out the form. Please include as much info as you can, this will make it a lot easier for me to fix it.
