Failure code meanings:

01: Unsupported by flash emulator. This means that the emulator (Ruffle) currently does not have the capability to run the game that has been submitted.

02: (for html5 games) I was unable to find an unblocked website that hosts that game, meaning I could still add it, but it would be blocked at most or all schools.

03: Unable to find game. This means I wasn't able to find that game anywhere on the internet.

04: This is not a web game. (For example Minecraft) and there is no possible way to add it to any website.
